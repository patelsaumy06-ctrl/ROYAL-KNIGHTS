# NeedLink AI 🤝

> Data-driven volunteer coordination for social impact — Smart Resource Allocation

## The Problem
Local NGOs collect field data through paper surveys and reports. This data is scattered, 
making it impossible to see where help is needed most or match volunteers efficiently.

## Our Solution
NeedLink AI uses **Gemini AI** to parse unstructured field reports and extract urgent needs 
automatically. A **geospatial matching engine** (Haversine distance) then connects the 
closest available volunteers to each task in real time.

## Tech Stack
- React 19 + Vite — Frontend
- Firebase Firestore — Real-time database
- Google Gemini AI — Document parsing & need extraction
- Leaflet + Google Maps — Geo-visualization
- Express.js — Secure backend API
